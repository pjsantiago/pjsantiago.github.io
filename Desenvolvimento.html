Plataforma Alvorada

A Plataforma Alvorada foi desenvolvida em 2015 para o hospital veterinário onde trabalho atualmente com o objetivo de modernizar fluxos que antes eram feitos em papel.

O sistema foi desenvolvido em C# WebForms, para Banco de Dados foi SQL Server e no FrontEnd foi utilizado o pacote do bootstrap.
A Plataforma roda em Intranet na empresa.
Atualmente estou desenvolvendo uma nova Plataforma, mais robusta e escalonável em C# .NET CORE MVC, Entity Framework 6 e bootstrap-5
